# FloatBall
自定义View仿360悬浮球与加速球

做成的效果有： 

1. 点击按钮后退出Activity，呈现一个圆形的悬浮球，可以随意拖动并会自动依靠到屏幕一侧，且拖动时会变成一张图片 

2. 当点击悬浮球时，悬浮球隐藏，底部出现一个加速球，双击加速球时，呈现水量逐渐增高且波动幅度较小的效果，单击时波浪上下波动且幅度渐小

3. 点击屏幕不包含底部加速球的部位，加速球会隐藏，悬浮球重新出现
